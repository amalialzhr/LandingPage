Madani

Madani Scholars merupakan program pengembangan kepemimpinan sebagai inkubator untuk mahasiswa berprestasi di Sumatera Barat. Program ini ditujukan untuk mahasiswa tahun ke-2 dan 3 untuk mengembangkan diri dan berkolaborasi dengan tujuan untuk membentuk rencana mereka selama masa kuliah serta mendorong untuk memaksimalkan potensinya. Hal ini sangat penting untuk dapat menghasilkan pemimpin muda yang dapat membawa inovasi dan perubahan positif untuk Sumatera Barat.
